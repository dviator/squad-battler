# Ideas — lightweight scratchpad

Drop unrefined game-dev thoughts here as plain markdown files. One idea per file,
named `kebab-case-slug.md`. **No status frontmatter, no ceremony** — this is a
notebook, not a tracker.

An idea has exactly two exits:

- **Graduates** → `/refine-idea` turns it into a `backlog/designs/<id>.md` and
  **deletes this file** (its content is carried into the design).
- **Discarded** → delete the file.

Optional frontmatter, if you feel like it:

```markdown
---
priority: 1   # 1 = do soon, 3 = someday
---

Whatever the thought is. A sentence is fine. A wall of text is fine.
```

If a thought is already fleshed-out enough to act on, it's fine to skip this
folder and write a design directly.
